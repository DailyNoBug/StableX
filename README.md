# StableX 三轴稳定器项目

使用STM32G431+DRV8313+AS5600作为稳定器关节，末端解算姿态后控制三轴电机旋转来实现稳定

目前项目仍处于工程机验证阶段，后续会针对结构进行优化改进，目前存在的问题是3d打印结构强度不够，电机扭矩不足。
